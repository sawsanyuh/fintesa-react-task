# Fintesa Transactions Dashboard

## Run locally
1. `npm install`
2. `npm run dev`
3. Open http://localhost:5173

## Tech stack
- React.js (Hooks)
- React Router v6
- Tailwind CSS
- Vite

## Notes
- Mock data located at `src/data/transactions.json`.
- Features: search by customer name/ID, status filter, sorting (date/amount), details page.
